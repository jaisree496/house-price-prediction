#  House Price Prediction

This project predicts house prices using Machine Learning.

##  Description
A regression-based model is trained on housing data to estimate prices based on input features.

##  Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Google Colab

## Model
Linear Regression

##  Accuracy
64.94%

##  Files
- housepriceprediction.ipynb → Main notebook
- model.pkl → Trained model

##  Steps
1. Data preprocessing
2. Feature selection
3. Model training
4. Prediction & evaluation
